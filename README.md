## Install dependencies

```bash
npm instal
```

## Config your credentials in `credentials.json`

```json
[
  {
    "type": "discord",
    "emailOrUsername": "username",
    "password": "password"
  },
  {
    "type": "twitter",
    "emailOrUsername": "username",
    "password": "password"
  }
]
```

## Run bot

```bash
npm run bot
```

> You can see number of tx via `http://localhost:3000`
